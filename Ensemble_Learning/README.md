# Ensemble Learning in Machine Learning

This project demonstrates various Ensemble Learning techniques used to improve model performance by combining multiple models.

## 📌 Techniques Covered

### 🔹 Boosting Methods
- AdaBoost
- Gradient Boosting
- XGBoost

### 🔹 Voting Method
- Hard Voting
- Soft Voting

### 🔹 Stacking Method
- Stacking Classifier

## 📂 Files

- adaboost.ipynb → AdaBoost implementation
- GradientBoosting.ipynb → Gradient Boosting implementation
- XGBoost.ipynb → XGBoost implementation
- VotingClassifier.ipynb → Voting ensemble model
- StackingClassifier.ipynb → Stacking ensemble model

## 🚀 What is Ensemble Learning?

Ensemble Learning is a technique where multiple machine learning models are combined to produce better predictive performance than individual models.
