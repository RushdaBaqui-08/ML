# Ensemble Boosting Methods

This project demonstrates implementation of popular boosting algorithms in Machine Learning.

## 📌 Algorithms Covered

- AdaBoost
- Gradient Boosting
- XGBoost

## 📂 Files

- adaboost.ipynb
- GradientBoosting.ipynb
- XGBoost.ipynb

## 🚀 Description

Boosting is an ensemble learning technique where multiple weak models are combined to create a strong predictive model. Each model focuses on correcting errors of previous models.

## ⚙️ Technologies Used

- Python
- Scikit-learn
- XGBoost
- Pandas
- NumPy
- Matplotlib / Seaborn

## ▶️ How to Run

1. Open Jupyter Notebook
2. Run each notebook step-by-step
