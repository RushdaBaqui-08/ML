# Support Vector Machines Implementation (SVC & SVR)

This project demonstrates the implementation of Support Vector Machines (SVM) using Python and Scikit-learn. It includes examples of both classification and regression using SVM models.

## 📁 Files Included

- svc_implementation.ipynb  
  Implementation of Support Vector Classifier (SVC) using the Iris dataset for classification.

- svr_implementation.ipynb  
  Implementation of Support Vector Regressor (SVR) using the Diabetes dataset for regression.

## 📊 Technologies Used

- Python
- Scikit-learn
- Pandas
- Jupyter Notebook

## 📌 Concepts Covered

- Support Vector Machines (SVM)
- Support Vector Classification (SVC)
- Support Vector Regression (SVR)
- Data preprocessing
- Feature scaling using StandardScaler
- Train-test split
- Model training and prediction
- Model evaluation

## 📈 Evaluation Metrics

Classification Model:
- Accuracy Score
- Classification Report

Regression Model:
- R² Score

## 🎯 Datasets Used

- Iris Dataset (for classification)
- Diabetes Dataset (for regression)

Both datasets are loaded from the `sklearn.datasets` library.

## 🚀 Purpose

The goal of this project is to understand how Support Vector Machines can be used for both classification and regression tasks using real datasets.
